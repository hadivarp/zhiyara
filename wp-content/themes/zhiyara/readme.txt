A complete Persian restaurant guide theme similar to Michelin Guide.

== Description ==

راهنمای رستوران‌های ژیارا - قالبی جامع برای معرفی و ارزیابی رستوران‌ها و کافه‌ها

ویژگی‌ها:
* سیستم امتیازدهی ستاره‌ای (۱-۵ ستاره)
* دسته‌بندی رستوران‌ها
* فیلتر پیشرفته
* طراحی RTL فارسی
* واکنش‌گرا و موبایل‌دوست

== Installation ==

1. Upload the theme files to `/wp-content/themes/zhiyara/`
2. Activate the theme through the 'Appearance > Themes' screen in WordPress
3. Configure the theme through the Customizer
